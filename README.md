# MetodoSimple-M

Método simple para resolver problemas paso a paso.

---

## Descripción

Este repositorio contiene una implementación sencilla del **método Simple M**, enfocada en entender el proceso más que en complicarlo.

La idea es:
- Resolver
- Entender
- Repetir

Sin sobrecargar con teoría innecesaria.
